# Restarunt-Ai-Automation
🤖 Just built an AI-powered food ordering agent using n8n!  The entire workflow: ✅ Customer chats → AI takes order (items + quantity) ✅ Confirms price breakdown automatically ✅ Asks customer name &amp; confirms order ✅ Appends order directly to Google Sheets in real time ✅ Reads FAQ &amp; Item List from sheets dynamically
